NegNasu.css

What is this ?

NegNasu.css is a CSS-stylesheet themed around the current 3.0 design of the australian purple wolf streamer Neg Nasu. This is an example page showing off all the available elements. No prerequisites, no images, just pure CSS.
Nassify your website today!

This project is a living work-in-progress.
Get the current version from github.com/madpew/negnasu-css.

Features:

  • colors, text-styles and font-definition
  • slide-headlines
  • lists
  • scrolling checkerboard
  • chatbubbles
  • sticky-notes
  • popovers
  • inputs

Below you can find a few examples. For more information consider looking at the source.

Checkerboard:

Use the class checkerboard on elements to give them the scrolling background.

Popovers:

Also known as popup or dialog are useful when you need to get the attention away from the main content. They can also be used in normal page content.

This is what popovers look like.
If you set the data-x property, the first button will become the x-button.
The title-text and color is also set using the properties data-type and data-title.
